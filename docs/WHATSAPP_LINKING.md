## English

### Why link accounts?

Linking your WhatsApp account to Telegram allows you to:
- Use credits purchased via Telegram Stars for WhatsApp transcriptions
- Manage your balance from a single Telegram account

### How to link

1. Open the bot in **Telegram**
2. Send `/link_whatsapp`
3. You will receive a 6-digit code (valid for 5 minutes)
4. Open the bot in **WhatsApp**
5. Send: `link XXXXXX` (replace with your code)
6. Done! Your accounts are now linked

### How to unlink

Send `/unlink_whatsapp` in Telegram.

### Notes

- Each Telegram account can be linked to one WhatsApp number
- Each WhatsApp number can be linked to one Telegram account
- Re-linking replaces the previous connection

## Русский

### Зачем привязывать аккаунты?

Привязка WhatsApp к Telegram позволяет:
- Использовать кредиты, купленные через Telegram Stars, для транскрипций в WhatsApp
- Управлять балансом из одного аккаунта Telegram

### Как привязать

1. Откройте бота в **Telegram**
2. Отправьте `/link_whatsapp`
3. Вы получите 6-значный код (действует 5 минут)
4. Откройте бота в **WhatsApp**
5. Отправьте: `link XXXXXX` (замените на ваш код)
6. Готово! Аккаунты привязаны

### Как отвязать

Отправьте `/unlink_whatsapp` в Telegram.

### Примечания

- К одному Telegram-аккаунту можно привязать один номер WhatsApp
- К одному номеру WhatsApp можно привязать один аккаунт Telegram
- Повторная привязка заменяет предыдущую

## Deutsch

### Warum Konten verknüpfen?

Die Verknüpfung deines WhatsApp-Kontos mit Telegram ermöglicht:
- Über Telegram Stars gekaufte Credits für WhatsApp-Transkriptionen nutzen
- Dein Guthaben über ein einziges Telegram-Konto verwalten

### So verknüpfst du

1. Öffne den Bot in **Telegram**
2. Sende `/link_whatsapp`
3. Du erhältst einen 6-stelligen Code (5 Minuten gültig)
4. Öffne den Bot in **WhatsApp**
5. Sende: `link XXXXXX` (ersetze durch deinen Code)
6. Fertig! Deine Konten sind verknüpft

### So trennst du die Verknüpfung

Sende `/unlink_whatsapp` in Telegram.

### Hinweise

- Jedes Telegram-Konto kann mit einer WhatsApp-Nummer verknüpft werden
- Jede WhatsApp-Nummer kann mit einem Telegram-Konto verknüpft werden
- Erneutes Verknüpfen ersetzt die vorherige Verbindung
